Proxy Suite Installer

One-command setup for Clash Verge Rev.
Manage proxy state across your terminal tools.

Terminal
$ curl -fsSL https://proxy.wusaqi.me/install.sh | bash

Requires bash, curl. Supports WSL / Linux / macOS.

Features

Unified CLI

A single proxy command to start, stop, test, and manage your proxy.

Tool Integration

Automatically configures git, npm, pnpm, yarn, and pip proxy settings.

Verified Downloads

SHA256 digest verification for all downloaded release assets.

Idempotent Install

Managed shell blocks prevent duplication. Reinstall anytime, no manual cleanup.

Daily Workflow

1 proxy on Start Clash Verge + enable proxy
2 proxy test Verify ports + outbound connectivity
3 proxy status Show compact state overview
4 proxy off Stop everything and clean env

Requirements

WSL / Linux bash curl awk sed grep cut
macOS bash curl hdiutil
